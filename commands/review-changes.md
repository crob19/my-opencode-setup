---
description: Review uncommitted changes
---

Review all uncommitted changes for quality issues.

Current changes:
!`git diff`

Untracked files:
!`git status --short`

Check for:
- Bugs or logic errors
- Security vulnerabilities
- Code quality issues (complexity, readability)
- Missing error handling
- Performance concerns
- Testing gaps
- TODO/FIXME comments that should be addressed

Suggest improvements before committing.
